## 1. Resumen del proyecto

En este proyecto se creó una libreria instalable en la cual se puede recorrer rutas e identificar archivos .MD, luego recorrerlos y analizar los links que están dentro de estos, extraerlos y entregar datos sobre estos, verificando cantidad, texto y si están funcionales o no.
